1、功能说明

	本示例展示如何从机器模式进入用户模式运行用户代码；

	RISC-V内核默认进入机器模式运行，通过修改MSTATUS位及mepc寄存器跳转入用户代码下运行；
	跳转代码见JumpUserMode
	示例中样例代码为LED D5的交替闪烁；

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    	IDE工具：NucleiStudio IDE for C/C++ 2021-02
	
	/* 开发板 */
	CM32M4xxR_LQFP128_STB

3、使用说明

	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        1、SystemClock：144MHz
        2、GPIO：PB1控制 LED D5闪烁
	3、开发板拨码开关配置：2 连接，其他断开；Boot0配置：LOW。

	使用方法：
		1.编译后下载程序复位运行；
	    	2.程序运行后，经进入用户模式，LED D5将间隔1秒交替闪烁；
		
4、注意事项

	进入机器模式的方法，将MSTATUS中MPP位清除，修改MEPC为用户程度入口，通过mret后进入
