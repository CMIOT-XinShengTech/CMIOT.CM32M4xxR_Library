1、功能说明

	/* 简单描述工程功能 */
        这个例程配置并演示CAN1，CAN2两个通道进行CAN报文收发的功能。

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2021-02
	
	/* 开发板 */
	CM32M4xxR_LQFP128_STB + 六类接口功能扩展板

3、使用说明
	
	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        SystemClock：144MHz
        USART：TX - PE8，RX - PE9，波特率115200。
	    CAN1：RX - PA11，TX - PA12，波特率500K。
		CAN2：RX - PB12，TX - PB13，波特率500K。

	/* 描述Demo的测试步骤和现象 */
        1.编译后下载程序复位运行；
        2.查看串口打印信息，CAN1发送报文到CAN2，CAN2发送报文到CAN1。当到打印信息为接收通过时程序运行正常；

4、注意事项
	需要在六类接口功能扩展板板上用跳帽将CAN1和CAN2的总线进行连接。
