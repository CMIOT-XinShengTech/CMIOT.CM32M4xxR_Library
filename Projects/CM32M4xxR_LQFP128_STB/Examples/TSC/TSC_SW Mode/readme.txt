1、功能说明
    1、轮询两个触摸按键SW4、SW5，有按键输入时LED闪烁

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2021-02
	
	/* 开发板 */
	CM32M4xxR_LQFP128_STB

3、使用说明
    
    系统配置；
        1、时钟源：HSE+PLL
        2、系统时钟：16MHz
        3、TSC端口： PA4:CH0       PA5:CH1 
        4、LED控制端口：PB0、PB1、PB2
		5、TIM3：重载计数，重载周期200us，产生中断。每隔10ms检查TIM2计数值，并切换TSC通道。
		6、TIM2：外部时钟模式2，TSCOUT作为TIM2 ETR进行计数。

    使用方法：
        1、编译后烧录到对应开发板，通电，有触摸按键时，通过LED闪烁指示按键状态
           SW4:D4闪烁一次
           SW5:D4闪烁两次
		2、串口输出对应的按键信息。
        
4、注意事项
     SW4和SW5不能焊接实体按键。
