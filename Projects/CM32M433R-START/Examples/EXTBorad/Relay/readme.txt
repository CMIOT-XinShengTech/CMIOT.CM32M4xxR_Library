1、功能说明

	展示继电器常开触点闭合和常闭触点闭合操作。

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START + CM32M433R-EXTEND

3、使用说明
	1、编译后下载程序并运行。
    2、运行RelayNOEnable后，常开触点闭合，听到“咔”声，万用表测量继电器(J8)1脚和2脚导通，1脚和3脚不导通。
    3、运行RelayNCEnable后，常闭触点闭合，听到“咔”声，万用表测量继电器(J8)1脚和2脚不导通，1脚和3脚导通。
    4、程序每隔1S进行常开触点和常闭触点的闭合切换，同时串口输出相应的信息。
4、注意事项
	无
