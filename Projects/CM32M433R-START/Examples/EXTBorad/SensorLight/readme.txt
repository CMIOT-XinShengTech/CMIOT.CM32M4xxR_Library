1、功能说明

	本例程展示芯片通过ADC检测光敏电阻大小，从而判断环境光纤强度

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START + CM32M433R-EXTEND

3、使用说明
	1、编译下载运行程序，可以从串口中看到ADC采样值、电压值、电阻值和环境光信息输出，输出间隔1s。
	2、将电阻靠近光源可以观察到电阻值变小，同时输出It is bright
	3、将电阻遮住光源可以观察到电阻变大，同时输出It is dark
		
4、注意事项
	无
