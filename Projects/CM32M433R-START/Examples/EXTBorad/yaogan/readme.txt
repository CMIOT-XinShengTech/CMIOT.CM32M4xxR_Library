1、功能说明

	观察板子上的摇杆所在位置，并显示在上位机软件上

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
	IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START + CM32M433R-EXTEND

3、使用说明
	1、下载程序
	2、修改tool文件夹下yaogan.py里对com口的设置
	3、通过命令行运行python yaogan.py
	4、转动摇杆，观察上位机软件里摇杆的位置显示
	
4、注意事项
	目前暂时不支持上位机界面的关闭，建议关闭命令行窗口以此关闭上位机界面
