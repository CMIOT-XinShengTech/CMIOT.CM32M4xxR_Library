1、功能说明

	本例程展示芯片控制温湿度传感器定时输出温度和湿度数据

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START + CM32M433R-EXTEND

3、使用说明
	1、编译下载运行程序，可以从串口中看到温度和湿度数据输出，输出间隔1s。
	2、向温湿度传感器吹气，可以看到输出温湿度数据的变化。
	3、温湿度传感器通过IIC进行控制，从机地址为AHT20_ADDRESS。

	无
