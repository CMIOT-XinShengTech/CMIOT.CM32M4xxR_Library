1、功能说明
    本例程展示了3个触摸按键和1个滑动触摸按键功能，使用TSC软件模式。

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2021-02
	
	/* 开发板 */
	CM32M4xxR_START + CM32M433R-EXTEND

3、使用说明
    1、编译后烧录到对应开发板，
    2、当按下3个触摸按键时，对应的扩展板LED亮起，松开触摸按键，LED熄灭。
    3、当手指滑动触摸按键，RGBLED亮起，并跟随滑动状态进行明暗变化，松开手指，RGBLED熄灭。
    4、当触摸按键状态变化时，串口会输出对应的检测信息。
        
4、注意事项
    关于TSC软件模式的程序原理，可以参考《AN0002-TSC功能及应用样例说明.pdf》
