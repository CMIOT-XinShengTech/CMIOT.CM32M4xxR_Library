1、功能说明

	本例程展示芯片控制6轴传感器定时输出角速度和加速度数据

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START + CM32M433R-EXTEND

3、使用说明
	1、编译下载运行程序，可以从串口中看到加速度数据和角速度数据输出，输出间隔1s。
	2、沿着X/Y/Z轴进行平移，可以看到加速度数据变化；沿着X/Y/Z轴进行转动，可以看到角速度数据变化。
	3、六轴传感器通过IIC进行控制，从机地址为QMI8658_SLAVE_ADDR_H
	4、关于六轴加速度的相关配置可以修改qmi8658.h中的宏进行控制，具体宏对应的含义，可以查阅六轴传感器相关手册。
