1、功能说明
	1、WWDG复位功能。


2、使用环境
	软件开发环境：  NucleiStudio IDE for C/C++ 2022-01
	硬件环境：      CM32M433R-START
3、使用说明
	
	系统配置；
		1、WWDG时钟源：PCLK1
		2、窗口值：42.7794 ms < n <58.2528ms
                3、指示灯：D6 D7
 		4、串口配置：
			- 串口直接使用Micro USB与 MCU 进行串口通信
                            - 数据位：8
                            - 停止位：1
                            - 奇偶校验：无
                            - 波特率： 115200               



	使用方法：
		烧录到评估板，上电后，指示灯D7不停的闪烁。说明窗口值刷新正常，代码正常运行。
                当把Delay(50)函数参数改成58以上，整个系统将一直处于复位状态。


4、注意事项
     无
