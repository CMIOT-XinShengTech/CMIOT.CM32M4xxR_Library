1、功能说明

	1、TIM1 利用更新中断，产生定时翻转IO

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START

3、使用说明

	系统配置；
		1、时钟源：
		            HSE=8M,PLL=144M,AHB=144M,APB1=36M,APB2=72M,TIM1 CLK=72*2M（APB Prescaler！=1）=144M
		2、中断：
		            TIM1 更新中断打开
		3、端口配置：
					PD3选择为IO输出
	    4、TIM：
		            TIM1使能周期中断
	使用方法：
		1、编译后打开调试模式，用示波器或者逻辑分析仪观察PD3的波形
	    2、程序运行后，TIM1的周期中断来临翻转PD3电平，周期1.09khz。

4、注意事项

	无
