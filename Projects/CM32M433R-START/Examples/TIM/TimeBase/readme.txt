1、功能说明

	1、TIM2 利用CH1 CH2 CH3 CH4 CC值产生定时中断，翻转IO电平

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START

3、使用说明

	系统配置；
		1、时钟源：
		            HSE=8M,PLL=144M,AHB=144M,APB1=36M,APB2=72M,TIM2 CLK=36M*2（APB Prescaler！=1）=72M
		2、中断：
		            TIM2 比较中断打开
		3、端口配置：
					PD3选择为IO输出
					PD4选择为IO输出
					PD5选择为IO输出
					PD6选择为IO输出
	    4、TIM：
		            TIM2输出冻结模式，CH1 CH2 CH3 CH4比较值产生中断。比较中断处理中，翻转PD3~PD6电平。
	使用方法：
		1、编译后下载程序，用示波器或者逻辑分析仪观察PD3、PD4、PD5、PD6的波形
	    2、程序运行后，对应通道达到比较值后，比较值累加并翻转对应IO口电平，输出波形频率分别为25/50/70/100Hz。
