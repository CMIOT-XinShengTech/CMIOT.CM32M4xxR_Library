1、功能说明

	示例演示如何计算输入信号的快速傅里叶变换，并输出最大能量及所在位置；
	输入数据为10KHz叠加噪声信号，采样点为2048；

2、使用环境

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    	IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START

3、使用说明

	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        1、SystemClock：144MHz
        2、USART配置如下：
                - 波特率 = 115200 baud
                - 字长 = 8数据位
                - 1停止位
                - 校验控制禁用
                - 硬件流控制禁用（RTS和CTS信号）
                - 接收器和发送器使能
		- UART4_Tx.PD0
                - UART4_Rx.PD1
        3、通过Micro USB连接开发板，确保J6、J7跳帽短接；或使用外部USB Serial工具连接TX4,RX4管脚；


	使用方法：
		1.编译后下载程序复位运行；
	    	2、程序运行后，将实现对输入信号进行傅里叶变换获得频域信号，通过log串口打印最大能量的数据及偏移，
		并与期望数据比对，比对成功输出Pass
