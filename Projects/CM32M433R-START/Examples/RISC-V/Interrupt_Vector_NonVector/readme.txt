1、功能说明

	本示例展示ECLIC的向量中断及非向量中断，示例中共实现了TIM2/TIM3两个通用定时器的普通定时中断，超时时间分别为1秒，0.5秒；
	其中TIM2超时中断为非向量中断，优先级为1，并在超时后将翻转LED D6，
	其中TIM3超时中断为向量中断，优先级为2，并在超时后将翻转LED D7;

3、使用说明

	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        1、SystemClock：144MHz
        2、GPIO：PB5、PB6、PB7 控制 LED(D6、D7、D8) 闪烁
	3、Boot0配置：LOW。
	4、TIM2定时器为超时中断，超时周期为1s，非向量中断，优先级为1；
	5、TIM3定时器为超时中断，超时周期为0.5s，向量中断，优先级为2；	
	
	使用方法：
		1.编译后下载程序复位运行；
	    	2.程序运行后，LED D6/D7将分别以1秒和0.5秒间隔进行闪烁，
		
4、注意事项

	向量中断通常不支持叶子函数调用，向量中断响应函数前添加__INTERRUPT
