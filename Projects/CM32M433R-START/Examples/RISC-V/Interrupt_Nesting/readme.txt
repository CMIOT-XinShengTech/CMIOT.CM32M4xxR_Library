1、功能说明

	本示例展示ECLIC中断嵌套特性，示例中共实现了TIM2/TIM3/TIM4三个通用定时器的普通定时中断，超时时间分别为5秒，0.5秒， 0.25秒；
	其中TIM2超时中断为非向量中断，优先级为1，并在超时后将翻转LED6，并延时1秒等待其他定时器中断；
7	其中TIM3超时中断为向量中断，优先级为2，并在超时后将翻转LED7;
	其中TIM4超时中断为非向量中断，优先级为3，并在超时后将翻转LED8;

	/* 软件开发环境：当前工程使用的软件工具名称及版本号 */
    	IDE工具：NucleiStudio IDE for C/C++ 2022-01
	
	/* 开发板 */
	CM32M433R-START

3、使用说明

	/* 描述相关模块配置方法；例如:时钟，I/O等 */
        1、SystemClock：144MHz
        2、GPIO：PB5、PB6、PB7 控制 LED(D6、D7、D8) 闪烁
	3、Boot0配置：LOW。
	4、TIM2定时器为超时中断，超时周期为5s，非向量中断，优先级为1；
	5、TIM3定时器为超时中断，超时周期为0.5s，向量中断，优先级为2；	
	6、TIM4定时器为超时中断，超时周期为0.25s，非向量中断，优先级为3；

	使用方法：
		1.编译后下载程序复位运行；
	    	2.程序运行后，LED6/D7/D8将分别按照定时器的周期交替闪烁；
		
4、注意事项

	向量中断通常不支持中断嵌套，若需要支持嵌套则需要在中断响应函数中添加 SAVE_IRQ_CSR_CONTEXT() /RESTORE_IRQ_CSR_CONTEXT()
