1、功能说明
	1、通过EXTI线来触发日历打印输出。


2、使用环境

	软件开发环境：  NucleiStudio IDE for C/C++ 2021-02
	硬件环境：      CM32M433R-START


3、使用说明
	
	系统配置；
		1、RTC时钟源：LSE
                2、中断线：EXTI_LINE5
		3、串口配置：
			- 串口为UART4（TX：PD0  RX：PD1）
			- 数据位：8
			- 停止位：1
			- 奇偶校验：无
			- 波特率： 115200 


	使用方法：
		编译后烧录到评估板，PC13连接到PF5口，上电后，串口每隔1S会打印一次对应日历时间


4、注意事项
	无
