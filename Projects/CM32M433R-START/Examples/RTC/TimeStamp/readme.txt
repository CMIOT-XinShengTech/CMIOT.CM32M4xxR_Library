1、功能说明
	1、通过EXTI线来触发时间戳。



2、使用环境

	软件开发环境：  NucleiStudio IDE for C/C++ 2021-02
	硬件环境：      CM32M433R-START


3、使用说明
	
	系统配置；
		1、时钟源：LSI
                2、EXTI中断IO：PD8
		3、串口配置：
			- 串口为UART4（TX：PD0  RX：PD1）
			- 数据位：8
			- 停止位：1
			- 奇偶校验：无
			- 波特率： 115200 


	使用方法：
		编译后烧录到评估板，把PC13连接到PD8，上电后，串口每隔1S会打印时间戳输出。


4、注意事项
	无
